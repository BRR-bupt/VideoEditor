# VideoEditor

VideoEditor 是基于 vue 和 typeScript 开发的视频剪辑网站

## Support Browsers

- Google Chrome 92 or later
- Edge

## Technology stack

- VUE
- TypeScript
- NUXT
- Express
- Threejs
- Ccapturejs
- FFmpeg

## Thanks

基于 toshusai 的开源项目 vega（MIT）做出的开发，开发包括但不限于：

- 使用了 express 服务代替 nuxt 服务，达到添加 resHeader 的效果
- 增加了 three 渲染缩放功能
- 修改了 duration 的 bug
- 修改了切换 asset 的 bug
- 删去了 electron 功能
- 进行了全面的中文注释
